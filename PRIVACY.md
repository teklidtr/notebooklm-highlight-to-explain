# Privacy Policy

NotebookLM Highlight to Explain is designed to run locally in your browser.

## Data Collection

The extension does not collect, sell, or share personal information.

## NotebookLM Content

When you highlight text in NotebookLM and choose Explain, Example, or Summarize, the extension reads that selected text in the current browser tab so it can build a prompt. The prompt is inserted into the NotebookLM chat panel and remains editable before you send it.

The extension does not send highlighted text, source titles, prompts, or chat content to any server controlled by the developer.

## Clipboard

If the extension cannot find the NotebookLM chat input, it may copy the generated prompt to your clipboard as a fallback.

## Storage and Analytics

The extension does not use analytics, tracking pixels, cookies, remote logging, or extension storage.

## Site Access

The extension is scoped to:

```text
https://notebooklm.google.com/*
```
