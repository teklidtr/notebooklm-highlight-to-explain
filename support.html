NotebookLM Highlight to Explain Support

NotebookLM Highlight to Explain is a Chrome extension for turning selected NotebookLM text into ready-to-send explain, example, and summarize prompts.

What it does

  • Opens a small popup from highlighted NotebookLM text.
  • Adds explain, example, or summarize prompts to the NotebookLM chat panel.
  • Includes the source title when the highlight comes from a NotebookLM source.

Customize Prompts

Configure the default prompt templates. You can use the placeholders {source} (for the source document title or page context) and {selection} (for the highlighted passage text) to customize the exact prompt layout and structure.

All changes auto-saved to sync storage.
