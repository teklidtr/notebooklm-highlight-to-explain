# Security Policy

## Supported Versions

Security fixes are handled for the latest public version of NotebookLM Highlight to Explain.

## Reporting a Vulnerability

Please report security issues privately by email:

```text
teklidtr@gmail.com
```

Include:

- A short description of the issue.
- Steps to reproduce it.
- The Chrome version and operating system you tested with.
- Any screenshots or logs that help explain the behavior.

Please do not open public GitHub issues for vulnerabilities until the issue has been reviewed.
